laka_vulkan
==== 

为Vulkan添加Cpp封装.

## 开源协议

zlib/libpng License.</br>
大致意思就是可以随便用,但不能剽窃,不能说是自己原创的.

## 目标

不省略功能</br>
        严格参照官方手册,对所有功能都提供使用途径.

使用方便</br>
        处理生成依赖管理生命周期,减少API调用参数,能自动完成的就让它自动完成.

易上手</br>
        丰富的信息输出,引导性的接口设计,避免出错的措施.
